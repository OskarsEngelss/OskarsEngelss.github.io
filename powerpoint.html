1. Kas un kāpēc ir HTML, CSS un Javascript?

HTML, CSS un Javascript ir rīki, ar ko veido mājaslapas. HTML un pati mājaslapa, CSS iedot tai stilu un Javascript veic komandas.

2. Kāda atšķirība starp front-end un back-end?

Front-end ir mājaslapas programmēšana, tas ko redz lietotājs, bet back-end ir serveri.

3. Kāda ir vienkāršākā HTML lapa?

Vienkāršākā HTML lapa ir bez CSS un Javascript koda.

4. Kas ir IDE?

IDE ir lietojumprogramma, kas palīdz programmētājiem efektīvi izstrādāt programmatūras kodu

5. Kas ir reaģējoša tīmekļa vietne?

Reaģējoša tīmekļa vietne ir vietne, kura maina izskatu attiecoties pret lietotāja ierīces ekrāna lielumu.

6. Kādi ir CSS selektori?

Ir četri CSS selektori

  • Vienkāršie selektroi (piem. ID)
  • Kombinatora selektori
  • Pseidoklases selektori
  • Pseidoelementu selektori

7. Kas ir box-model?

Box-model ir tas, kas aptver katru HTML elementu.

8. Kā centrēt HTML elementus, izmantojot FlexBox?

Izmantojot:
display: flex;
justify-content: center;
align-items: center;
